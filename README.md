Project Overview

The Interactive Story Book is a web-based recreation inspired by the classic childrenâ€™s storytelling toy experience. Users can create silly or adventurous stories by selecting different word combinations or generating a random story with one click. Features
Core Features

Manual story creation by selecting 5 different word parts

Random story generator (one-click auto story)

Reset button to start over

Two story modes:

Funny Story Mode

Adventure Story Mode

Text output display

Audio output using browser Text-to-Speech (SpeechSynthesis API)

Enhanced User Experience

Bright, playful UI inspired by classic storytelling toys

Button cycling interaction

Instant audio playback of generated stories

Mode switching with reset functionality
